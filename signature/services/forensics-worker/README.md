# forensics-worker 板块说明

## 板块内容
- Python 抓包工作进程。
- 根据任务参数调用 `tshark` 输出 `pcap`。
- 计算文件 `sha256` 和大小，**回写 forensics 表元数据**。

## 边界
- 只负责取证执行。
- 不负责审批、策略判定、事件分析。

## 对外接口
- 输入参数：`forensics_id`、抓包窗口、过滤表达式、输出路径。
- 输出产物：`pcap` 文件 + 摘要信息（hash/size）+ **数据库状态更新**。

## 关键函数
- `src/capture.py`：抓包执行入口与文件摘要计算、数据库回写。
