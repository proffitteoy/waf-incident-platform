# 前端接口对齐情况

> 最后更新：2026-03-23（已按 `docs/api-spec.md` 全量对齐）

## 对齐汇总

- 已接线（路径已对齐）：2 个页面
- 未接线（使用本地假数据，待后续接入）：15 个页面

---

## API 模块变更说明

| 文件 | 变更内容 |
|------|----------|
| `http.js` | 新增：请求时自动注入 `Authorization: Bearer <token>` |
| `user.js` | `POST /user/login` → `POST /auth/login`；移除 register；新增 `getMe()` → `GET /me` |
| `incidents.js` | 保留 list/detail；`addComment` 路径改为 `/incidents/:id/comment`（单数）；`triggerAnalyze`/`analyzeBatch`/`updateStatus` 替换为 `analyzeEvents` → `POST /incidents/analyze-events`；`createAction` 拆为 `executeAction` → `POST /incidents/:id/actions/execute` 和 `requestApproval` → `POST /incidents/:id/actions/request-approval`；新增 `getLlmReports`/`createLlmReport` → `GET|POST /incidents/:id/llm-reports` |
| `actions.js` | 仅保留 `rollback` → `POST /actions/:id/rollback`；移除 list/detail/execute/status/logs |
| `approvals.js` | 保留 list/approve/reject；移除 detail/batchDecide |
| `forensics.js` | `list` → `listByIncident(incidentId)` → `GET /incidents/:id/forensics`；`create` → `capture(incidentId)` → `POST /incidents/:id/forensics/capture`；新增 `updateStatus` → `PATCH /forensics/:id/status`；移除 cancel/status(GET)；保留 detail/download |
| `policies.js` | 保留 list/update；移除 detail/create/remove/updateStatus |
| `llmReports.js` | 仅保留 `listByIncident` → `GET /incidents/:id/llm-reports`；移除 list/detail/regenerate |
| `dashboard.js` | **新增**：`overview` → `GET /dashboard/overview`；`timeseries` → `GET /dashboard/timeseries` |
| `events.js` | **新增**：`listEvents` → `GET /events`；`listAlerts` → `GET /alerts` |
